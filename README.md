# Personal Portfolio

A personal portfolio website built with React, TypeScript, Tailwind CSS, and Framer Motion. It showcases who I am, what I build, and what I've learned — all in one place.

🔗 **Live Site:** [View Portfolio](https://dev-chadan24.github.io/PORTFOLIO)

---

## What's Inside

- **Hero** — Quick intro, role, and call-to-action
- **About** — A bit about me and my background
- **Skills** — Tech I work with daily (Python, React, and more)
- **Projects** — Real things I've built, with links
- **Experience** — Work and internship history
- **Education** — Academic background

---

## Tech Stack

| Tech | Why |
|---|---|
| React + TypeScript | Component-based UI with type safety |
| Tailwind CSS | Fast, utility-first styling |
| Framer Motion | Smooth animations and transitions |
| Vite | Fast dev server and build tool |

---

## Run Locally

```bash
# Clone the repo
git clone https://github.com/dev-chadan24/PORTFOLIO.git

# Move into the folder
cd PORTFOLIO

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

---

## Build for Production

```bash
npm run build
```

Output goes into the `dist/` folder.

---

## Deploy to GitHub Pages

```bash
npm run deploy
```

Make sure `gh-pages` is installed and the `homepage` field is set in `package.json`.
